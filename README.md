# docs.alpaka.space

Documentation for Jugend hackt Infra

## getting started

```
python3 -m venv venv
venv/bin/activate
pip3 install -r requirements.txt
```

## run mkdocs locally

`mkdocs serve` & view `http://127.0.0.1:8000`


## deploy mkdocs

Create a MR into the `main` branch.

It'll build automatically.

**DON'T DO `mkdocs gh-deploy`!**
